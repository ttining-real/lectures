## 13. 콜백 함수

### 13-1. 콜백 함수란?

- 콜백 함수는 다른 함수의 인수로 전달되어, 해당 함수에 의해 호출되는 함수를 의미한다.
- 콜백 함수는 비동기 작업이나 특정 동작 후 실행될 코드를 정의할 때 자주 사용된다.

<br>

#### 📌 특징

- 콜백 함수는 호출되지 않고 전달만 된다. 실행은 해당 콜백을 받는 함수에 의해 제어된다.
- 동기 및 비동기 작업 모두에 사용된다.

#### 📌 예시

```javascript
function main(callback) {
  console.log("main 함수 실행");
  callback(); // 전달받은 함수 호출
}

function sub() {
  console.log("sub 함수 실행");
}

main(sub); // sub: 콜백 함수
```

**결과**

```
main 함수 실행
sub 함수 실행
```

**✅ 설명**

- `sub` 함수는 `main` 함수의 매개변수로 전달된 뒤, `main` 내부에서 호출된다.
- 이때 `sub` 함수는 콜백 함수로 동작한다.

---

### 13-2. 콜백 함수의 활용

- 콜백 함수는 다양한 상황에서 활용된다.
- 특히, 비동기 작업(예: 타이머, 이벤트 핸들링, 데이터 처리)에서 중요한 역할을 한다.

<br>

#### 📌 타이머에서의 활용

```javascript
setTimeout(() => {
  console.log("1초 후 실행");
}, 1000);
```

**✅ 설명**

- `setTimeout`은 두 번째 인수로 콜백 함수를 받는다.
- 1초 후 이 콜백 함수가 실행됩니다.

<br>

#### 📌 배열 메서드에서의 사용

```javascript
const numbers = [1, 2, 3, 4, 5];
numbers.forEach((num) => {
  console.log(num * 2);
});
```

**결과**

```
2
4
6
8
10
```

**✅ 설명**

- `forEach` 메서드는 콜백 함수를 각 배열 요소에 대해 호출한다.
- 이 예제에서는 배열의 각 요소를 2배로 만들어 출력한다.

---

### 13-3. 요약

| 구분        | 예제제                                           | 설명                                  |
| ----------- | ------------------------------------------------ | ------------------------------------- |
| 타이머      | `setTimeout(() => console.log("실행"), 1000)`    | 일정 시간 후 실행할 작업 정의         |
| 배열 메서드 | `numbers.forEach((num) => console.log(num * 2))` | 각 배열 요소에 대해 특정 작업 수행    |
| 일반 사용   | `main((callback) => callback())`                 | 다른 함수 실행 후 특정 로직 실행 가능 |
