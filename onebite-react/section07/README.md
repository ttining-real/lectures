**컴포넌트의 라이프사이클**

```
모든 리액트 컴포넌트는 라이프 사이클을 가진다.
```

- `Mount` : 컴포넌트 첫 렌더링
- `Update` : 컴포넌트 리-렌더링
- `UnMount` : 컴포넌트 렌더링 제외(사라짐) => 클린업
