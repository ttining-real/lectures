## 1. Context란

### 📍 React Context

- 컴포넌트 간의 데이터를 전달하는 또 다른 방법
- 기존의 `Props`가 가지고 있던 단점을 해결할 수 있다.

<br>

#### 📌 `Props`의 단점

- 부모 → 자식으로만 데이터를 전달할 수 있다.
- Props Drilling

<br>

#### 📌 `Context`

- 데이터 보관소 (객체)
- Props Drilling을 해결할 수 있다.
