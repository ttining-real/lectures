## script

> 브라우저 객체 모델 이해하기
> Browser Object Model

<br />

### `window.load`

HTML 파싱 DOM 생성 그리고 외부 콘텐츠(images, script, css, etc)가 로드된 후 발생하는 이벤트

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script>
      window.onload = function () {
        const btn = document.querySelector("#btn");

        console.log("btn: ", btn);

        btn.addEventListener("click", function () {
          alert("Hello World!");
        });
      };
    </script>
    <title>Document</title>
  </head>
  <body>
    <button id="btn">버튼</button>
  </body>
</html>
```

<br />

### `DOMContentLoaded`

HTML 파싱 DOM 생성 후 발생하는 이벤트

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script>
      document.addEventListener("DOMContentLoaded", function () {
        const btn = document.querySelector("#btn");

        console.log("btn: ", btn);

        btn.addEventListener("click", function () {
          alert("Hello World!");
        });
      });
    </script>
    <title>Document</title>
  </head>
  <body>
    <button id="btn">버튼</button>
  </body>
</html>
```

### `defer` 속성

HTML 파싱과 함께 비동기로 JavaScript 파일을 불러온다.

- HTML 파싱이 완료된 후, JavaScript 코드를 실행한다.

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- HTML 파싱이 완료된 후, JavaScript 코드를 실행한다. -->
    <script src="./04.script.js" defer></script>
    <title>Document</title>
  </head>
  <body>
    <button id="btn">버튼</button>
  </body>
</html>
```

<br />

### `async` 속성

HTML 파싱과 함께 비동기로 JavaScript 파일을 불러온다.

- HTML 파싱이 완료되지 않았더라도, 먼저 로딩되는 JavaScript 파일부터 실행된다.
- JavaScript 파일을 실행할 때는 HTML 파싱이 중단된다.

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- HTML 파싱이 완료되지 않았더라도, JavaScript 코드를 실행한다. -->
    <script src="./04.script.js" async></script>
    <title>Document</title>
  </head>
  <body>
    <button id="btn">버튼</button>
  </body>
</html>
```
