## 3. 프로젝트 세팅

- 프로젝트 설치

  ```bash
  pnpm create vite@latest # react app 설치
  ```

- `.prettierrc.json` 파일 생성 및 작성
  ```json
  {
    "trailingComma": "all",
    "singleQuote": true,
    "tabWidth": 2,
    "semi": true
  }
  ```
- 플러그인 설치
  ```bash
    pnpm add -D eslint-plugin-simple-import-sort # 플러그인 설치
  ```
  - `eslint.config.js` 파일 내용 추가 : `plugins`, `rules`
