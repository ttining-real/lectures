## 1. SurveyPie 소개

### 1-1. 프로젝트 목표 🎯

- **SurveyPie** 라는 설문조사 서비스와 **Admin** 개발
- 프론트엔드 개발 실무의 이해
- React 개발 과정의 다양한 고민 이해

<br>

### 1-2. SurveyPie 서비스

- 구글 폼과 유사한 설문조사 서비스
- 한 페이지에 하나의 문항을 입력받는다.

<br>

### 1-3. SurveyPie 기술 스택

- `recoil` : 전역 상태 관리
- `axios` : API 연동
- `react-router` : 라우팅
- `styled-components` : 스타일링
- `immer` : 데이터 불변성 유지
